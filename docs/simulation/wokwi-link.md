## Create Your Own Simulation

1. Go to [https://wokwi.com](https://wokwi.com)
2. Click **New Project**
3. Select **Arduino UNO**
4. Follow the steps in `simulation-guide.md`
5. After building your simulation, click **Share** and paste the link here

## Limitations Reminder

| What you can simulate | What you must test on hardware |
|---|---|
| DS18B20 sensor reading | 433 MHz RF communication |
| I2C LCD display output | RF range testing |
| OneWire protocol | VirtualWire timing |
| Temperature formatting logic | Real-world interference |
